1 Создание web-сайта (часть 1 - HTML)

  1. Выбрать, скачать, установить и запустить среду разработки
  2. Вам нужно создать «сайт» в этой среде
  3. Создать веб-страницу (т.е. одну HTML страницу). Это будет страница вашего веб-портала
  4. Приготовьте файл с картинкой для страницы вашего портала
  5. Прикрепите изображение к странице портала
  6. Создайте одну или две дополнительных страницы вашего сайта. Их содержание может быть любым и должно отражать ваши интересы.
  7. Создайте гиперссылки на страницы портала
  8. Создайте гиперссылку на сайт dvfu.ru. Сайт ДВФУ должен открываться всегда в новой вкладке браузера.
  9. Скопируйте ваши файлы страниц и картинку на web-сервер. Она должна быть доступна всем желающим в интернете. Рекомендуется в качестве хостинга использовать Github Pages

2 Создание web-сайта (часть 2 - CSS, JavaScript)

  1. Создать веб-страницу (т.е. одну HTML страницу) с названием elka.html. Это будет страница – часть вашего веб-портала
  2. Ссылку на эту страницу добавить в меню файла index.html из лабораторной 1
  3. Приготовьте файл с картинкой минимум 4 типов елочных игрушек и картинку елки. Они могут отличаться цветом, формой и размером. Рекомендуется готовить картинки в формате png с включенным альфа каналом (вместо белого фона прозрачный)
  4. В части страницы создайте блок с типами игрушек. Поверх каждой из игрушек должна находиться иконка «+», дающая пользователю понять, что нажав на кнопку, на елку добавится новая игрушка
  5. Добавьте обработчики событий на каждую игрушку, чтобы по нажатии на картинку, на страницу добавлялась одна игрушка выбранного типа, в случайном месте на экране (в видимой области)
  6. Для каждой добавленной игрушки добавьте обработчик событий, отвечающий за перетаскивание игрушек курсором по странице
  7. Добавьте изображение елки на страницу
  8. Создайте обработчик событий для елки, чтобы при перетаскивании курсором (drag and drop) елка двигалась в нужном направлении. Если картинки добавленных игрушек находятся поверх елки, то они передвигаются вслед за елкой
  9. Добавьте кнопку «построить имя» на страницу рядом с кнопками добавления игрушек
  10. Добавьте обработчик событий кнопки, при нажатии на которую все добавленные игрушки упорядочиваются в форме Вашего имени. Имя формируется слева направо: если на страницу добавлено мало игрушек, имя сформируется не до конца

3 Введение в TeX

  1. Установить компилятор TeX. Рекомендуемые сборки: для windows – MikTeX; для mac – MacTeX, для linux – texlive
  2. Выбрать и установить текстовый редактор для написания tex – файла. Рекомендуется среда разработки TeXstudio (кроссплатформенная), но подойдет любой текстовый редактор
  3. Создать .tex документ. В документе описать минимальную разметку, указать класс документа article. В тексте документа указать свои фамилию, имя и отчество. Добиться безошибочной компиляции документа компилятором pdflatex (команда в консоли либо кнопка F5 в TexStudio). Подойдет и другой компилятор, напр. LuaLaTex, XeTeX, и др.
  4. После ФИО создать главу – название своей последней курсовой работы и любые два параграфа из той же работы
  5. Наполнить параграфы минимум тремя абзацами текста
  6. В текст добавить уравнение, после текста добавить рисунок
  7. Между параграфами добавить разрыв страницы
  8. В конце документа добавить автособираемое оглавление
  9. Добиться безошибочной компиляции документа компилятором pdflatex
  10. Загрузить pdf документ на сервер, создать на него ссылку на сайте
